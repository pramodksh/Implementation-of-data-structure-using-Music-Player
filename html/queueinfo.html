what is QUEUE?

A Queue is a linear structure which follows a particular order in which the operations are performed. The order is First In First Out (FIFO).

Basic operation on Queue

Enqueue

Adds an item to the queue. If the queue is full, then it is said to be an Overflow condition.

Dequeue

Removes an item from the queue. The items are taken in the same order in which they are inseted. If the queue is empty, then it is said to be an Underflow condition.
